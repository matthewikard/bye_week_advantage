# bye_week_advantage
Thanks to [spreadspoke](https://www.kaggle.com/datasets/tobycrabtree/nfl-scores-and-betting-data) for providing historical NFL spread data in a clean format. 

Analyzing NFL team success after a bye week using a Monte Carlo simulation based off historical game data. A more detailed explanation of this analysis can be found on [my substack](https://ikard.substack.com/p/the-bye-week-advantage).
